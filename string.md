# String

## Apuntes String

- Eliminar espacios Ej(" yes ")

```
>>>" yes ".strip()
'yes'

>>>" yes ".lstrip()
'yes '

>>>" yes ".rstrip()
' yes'
```

- Contar caracteres
```
>>>"La e se repite 6 veces".count("e")
6
```
- Determinar si una cadena termina en una determinada subcadena
```
>>>"pineapple".endswitch("apple")
True
```
